# RECOVERI — Универсальный советник для MT5

Советник для **«разруливания» счёта из минуса**: берёт под управление существующие позиции (свои, ручные или все подряд) и закрывает их корзиной по выбранной стратегии восстановления.

## Установка

1. Скопируйте файл `RECOVERI.mq5` в каталог `<MT5>/MQL5/Experts/`.
2. В MetaEditor нажмите **Compile** (F7).
3. В терминале MT5 откройте нужный график, перетащите советника на график.
4. Во вкладке **Common** разрешите автоторговлю.

## Стратегии восстановления (`InpMode`)

| Режим | Описание |
|------|----------|
| `0 TargetProfit` | Просто ждёт, пока сумма по корзине достигнет цели, затем закрывает всё. Новых сделок не открывает. |
| `1 Averaging` | Усреднение в сторону убытка (одинаковые лоты или с линейной прибавкой). |
| `2 MartingaleGrid` | Мартингейл-сетка с множителем лота и шага. |
| `3 HedgeLock` | Локирование встречной позицией при превышении порога убытка. |
| `4 SmartClose` | Парное закрытие: самая прибыльная + самая убыточная, если суммарно ≥ цели. |

## Какие позиции брать в управление

- `InpManageScope`:
  - `0 ALL` — все позиции
  - `1 MANUAL` — только ручные (magic == 0)
  - `2 OWN` — только сделки этого советника (по `InpMagic`)
- `InpSymbolScope`:
  - `0 CURRENT` — только символ графика
  - `1 ALL` — все символы счёта (только для `TargetProfit` и `SmartClose`, без доливок)

## Цель закрытия корзины

- `InpTargetType`:
  - `0 MONEY` — в валюте счёта
  - `1 PERCENT` — в % от баланса
  - `2 PIPS` — эквивалент в пунктах (по средневзв. лоту)
- `InpTargetProfit` — значение цели.
- Опционально включается **трейлинг корзины** (`InpUseBasketTSL`), фиксирующий часть прибыли после её роста.

## Защита

- `InpUseEquityStop` + `InpEquityStopPct` — аварийное закрытие, если equity упало ниже % от баланса.
- `InpCloseOnly` — режим «только закрытие»: советник не открывает новых сделок, только ждёт цель и обслуживает трейлинг.
- `InpMaxTrades`, `InpMaxLot`, `InpMaxSpreadPts` — предохранители.

## Типовые сценарии

**1) «Закрой мне руками открытые сделки в плюсе $10 и больше ничего не делай»**
```
InpMode          = 0 (TargetProfit)
InpManageScope   = 1 (MANUAL)
InpSymbolScope   = 1 (ALL)
InpTargetType    = 0 (MONEY)
InpTargetProfit  = 10.0
InpCloseOnly     = true
```

**2) «Усредни мою убыточную ручную позицию по EURUSD до безубытка»**
```
Открыть советника на графике EURUSD
InpMode          = 1 (Averaging)
InpManageScope   = 1 (MANUAL)
InpSymbolScope   = 0 (CURRENT)
InpStepPoints    = 300
InpStepMultiplier= 1.2
InpStartLot      = 0.01
InpMaxTrades     = 8
InpTargetProfit  = 5.0
```

**3) «Залочь мне минус, пока не разберусь»**
```
InpMode            = 3 (HedgeLock)
InpLockTriggerLoss = 50.0   (лочим, если -50 и хуже)
InpLockLotFactor   = 1.0    (полное локирование)
```

**4) «Подбери из открытых пары, чтобы хоть по копейке выйти»**
```
InpMode         = 4 (SmartClose)
InpManageScope  = 0 (ALL)
InpSymbolScope  = 1 (ALL)
InpTargetProfit = 0.5
```

## Информационная панель

При `InpShowPanel = true` в левом верхнем углу графика отображаются: режим, область управления, число позиций, BUY/SELL объёмы и средневзвешенные цены, текущий P/L корзины, пик и цель, статус.

## Дисклеймер

Все методы восстановления (усреднение, мартингейл, локирование) увеличивают риск потери счёта. Используйте на демо или с минимальными лотами и обязательным `InpUseEquityStop`. Автор ответственности не несёт.
